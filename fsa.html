Apa itu Finite State Automata?

Finite State Automata (FSA) adalah model komputasi sederhana yang terdiri dari sejumlah state (kondisi) dan aturan perpindahan (transisi) antar state berdasarkan simbol yang dibaca satu per satu.

DFA (Deterministic Finite Automata) - setiap state hanya punya tepat satu tujuan untuk setiap simbol input, sehingga jalannya pasti dan tidak bercabang.

NFA (Non-deterministic Finite Automata) - satu state boleh punya lebih dari satu kemungkinan tujuan untuk simbol yang sama, sehingga mesin bisa "berada di beberapa state sekaligus".

Sebuah string dikatakan diterima jika, setelah seluruh simbolnya dibaca habis, mesin berhenti di salah satu state akhir (F).

Cara menggunakan modul ini
  1. Isi alfabet (simbol yang dipakai, misal 0,1) dan daftar state (misal q0,q1,q2).
  2. Klik "Buat tabel transisi" untuk memunculkan tabel pengisian.
  3. Pilih state awal, lalu centang state akhir (boleh lebih dari satu).
  4. Isi tabel: untuk setiap state dan setiap simbol, tulis mau pindah ke state mana. Untuk NFA, boleh tulis lebih dari satu tujuan dipisah koma.
  5. Ketik string yang ingin diuji, lalu klik "Jalankan simulasi".
  6. Hasil, jejak transisi, definisi formal, dan diagram visual akan muncul di bawahnya.
